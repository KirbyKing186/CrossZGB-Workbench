# CrossZGB-Workbench
A collection of experimental projects made with the CrossZGB engine.

## Medamayaki
A platformer project inspired by "Palm Tree Paradise" of Wario Land 4. This project demonstrates many features, such as advanced platformer controls and a dynamic map system stored in SRAM.
